# 🔍 Lost & Found System

A full-stack Lost & Found web application that helps users report lost items, post found items, and connect with others through real-time communication.

---

## 🚀 Features

- 🔐 User Authentication (JWT Based)
- 📦 Post Lost & Found Items
- 🔍 Smart Matching System (Lost ↔ Found)
- 💬 Chat System (User to User Communication)
- 📁 Image Upload Support
- 🗂️ Organized Backend (MVC Structure)

---

## 🛠️ Tech Stack

### Frontend:
- HTML
- CSS
- JavaScript

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB (Mongoose)

### Other Tools:
- JWT Authentication
- Multer (for file uploads)
